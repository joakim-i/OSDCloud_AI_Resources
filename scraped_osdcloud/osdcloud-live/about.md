# About OSDCloud Live

OSDCloud Live (osdcloud.live) is a GitHub Repository used for testing OSD related PowerShell scripts easily, without having to install a PowerShell Module and related dependencies.

The following endpoints are available for use

* [Test OSDCloud requirements using your WinPE Boot Image](/osdcloud-live/about/test.osdcloud.live.md)
* [Deploy OSDCloud using your existing WinPE Boot Image](/osdcloud-live/about/deploy.osdcloud.live.md)
* [Export third-party drivers from your existing Windows 11 PC](/osdcloud-live/about/exportoem.osdcloud.live.md)
